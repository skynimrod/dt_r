. 1. par() ----  设置或查询 图形参数. 

     par( ..., no.readonly = FALSE )
     <highlevel plot> (..., <tag> = <value> )

      tag 表中包含参数的内容。

      如果par() 没有参数或 par(no.readonly=TRUE), 那么表示查询所有的图形参数.

     以下参数只读， 不能被设置:

       "cin", "cra","csi", "cxy", "din", "page"

     有些参数只能通过调用par() 来设置:

       "ask"

       "fig", "fin"

       "lheight"

       "mai", "mar", "mex", "mfcol", "mfrow", "mfg"

       "new"

       "oma", "omd", "omi"

       "pin", "plt", "ps", "pty"

       "usr"

       "xlog", "ylog"

       "ylbias"

    有些保留的参数也可以通过高级函数来设置, 例如： plot.default, plot.window, points, lines, abline, axis, title, text, mtext, symbols, arrows, polygon, rect, box, contour, filled.contour 和 image.

    无论如何, bg, cex, col, lty, lwd, pch 等参数通过plot 函数设置， 比图形参数要好的多。

   图形参数说明:

        adj :  决定text, mtext或title 中的文本内容的对齐方式. 0 左对齐, 0.5 中间对齐, 1 右对齐.

        ann :  如果是FALSE, 需要通过plot.default设置.

        ask :  逻辑值. 如果是TRUE, 则要求用户输入, 甚至非屏幕设备也可以用(不过就是没有效果:) )

        bg  :  背景颜色. background

        bty :  一个字符, 决定plots 外面box的类型。   "o"(缺省)，"l","7","c","u","]", "n"

              "o" 全封闭  "l" L型封闭  "7" 有2个角没封闭 "c"右面没封闭 "u" 上面没封闭 
              "]"左面没封闭(上下，保留坐标) "n"没有box, 并且坐标有缺口

        cex :  数字值.   放大倍数.

        cex.axis : axis 的放大倍数. ???

        cex.lab : x,y 标签的放大倍数

        cex.main: 主标题的放大倍数

        cex.sub : 子标题的放大倍数

        cin :  R.O.;  (宽，高) (width, height)， 类似还有cra

        col :  plot 的颜色.  

        col.axis : axis 的颜色, 缺省是"black"  黑色

        col.lab  : x, y 标签的颜色, 缺省是"black"

        col.main : 主标题的颜色, 缺省是"black"

        col.sub  : 子标题的颜色, 缺省是"black"

        cra :  R.O.;  缺省字符大小(width, height)， (像素)

        crt :  旋转角度.

        csi :  R.O.; 字符高度

        cxy :  R.O.; 字符大小，(width, height) 像素 

        din :  R.O.; 设备维度 (width, height)  inches

        err :  (没有实现)

        family : 字体名称. 最大200字节.

        fg  :  plot 前景(foreground)颜色

        fig :  数字向量  c(x1, x2, y1, y2 )  在设备上显示的范围. 

        fin :  figure 范围维度 (width, height), inches

        font : 字体

        font.axis : axis 字体

        font.lab : x, y 标签的字体

        font.main : 主标题的字体

        font.sub : 子标题的字体

        lab : 数字向量 c(x,y,len)   x,y轴(axis) 的长度以及刻度(tickmarks)。 缺省是(5,5,7)。  len 在R中没有实现

        las : 数字 {0,1,2,3}， 轴(axis） 标签的风格. 缺省是0(平行parallel)， 1(水平horizontal)，2(垂直perpendicular), 
 
                    3(垂直 vertical) ？？？ 2，3 的区别是什么？  3 似乎没有横坐标， 只需要纵坐标, 参见下面的例子

        lend : 线的风格。 设置以下内容:  0(缺省，圆弧 round),1(连接线 butt), 2(直角线square)

        lheight : 线高 ，缺省是1

        ljoin : 线相交的风格。  0 圆弧round   1 斜接(mitre)  2  斜角接 bevel
 
        lmitre : 线斜接的限制

        lty :  线类型  0=black  1=solid(缺省) 2=dashed  3= dotted  4=dotdash 5=longdash 6=twodash

        lwd :  线宽

        mai :  数字向量  c(bottom, left, top, right)  指定的是plot 区域在页面中的位置 距离下边长度， 距离左边长度等等

        mar :  数字向量 c(bottom, left, top, right) 距离plot 区域内四个边的距离, 缺省 c(5,4,4,2)+0.1

        mex :  字符大小， ？？？？ 

        mfcol, mfrow :  向量 c(nr, nc)  列 和行  ？？？

        mfg :   数字向量 c(i,j). 指示后面要画的数组  ？？？

        mgp :  边界线, 轴标题或轴标签或轴线. ??

        mkh :  符号高度。   -- 在R里被完全忽略了.

        new :  逻辑值， 缺省是FALSE.  如果是 TRUE , 那么下一个plot 不会清除现有内容, 因为它在新的设备上.

        oma :  向量 c(bottom, left, top, right)  文字距离边界线的长度

        omd :  向量 c(x1, x2, y1, y2) ??????

        omi :  向量 c(bottom, left, top, right) 外部边界的尺寸  inches

        page:  R.O.;  布尔值, 指示下一个调用是否 要用 plot.new 来启动一个新页. 

        pch :  单个符号或字符的大小.

        pin :  当前plot 的维度  (width, height)  inches

        plt :  向量 c(x1,x2, y1, y2) plot 区域与当前指定区域的 坐标

        ps  :  整数. 文本中点的大小.   1/72 inch

        pty :  字符。 用于指定plot 区域的类型.  's' 创建一个方形， 'm' 创建最大的plot 区域

        smo :  (没有实现) 

        srt :  字符旋转角度

        tck :  刻度长度(宽度)

        tcl :  高度坐标的刻度长度

        usr :  向量  c(x1, x2, y1, y2) plot 区域的坐标.

        xaxp : 向量 c( x1, x2, n ) 刻度的坐标。。。。？ 

        xaxs : x 坐标轴的风格。     只能设置"r"和"i", 其他"s","e","d" 都没有实现

        xaxt : 字符。 指定x轴的风格

        xlog : 逻辑值. 对一个新设备, 缺省是FALSE.  如果是TRUE, 使用对数比例(logarithmic scale)

        xpd  : 逻辑值或NA.FALSE  所有的plot 剪切到plot 区域。TRUE 所有 plot 剪切到指定区域. NA 所有Plot剪切到设备区域.

        yaxp : 向量 c(y1, y2, n) 刻度的坐标

        yaxs : y 坐标轴的风格

        yaxt : 字符， 指定y坐标轴的风格

        ylbias : 位置的真实值。 ？？

        ylog : 逻辑值, 参见  xlog 
   例子:

       op <- par(mfrow = c(2, 2),                            # 2 x 2 pictures on one plot
                  pty = "s")                                 # square plotting region,
                                                             # independent of device size

       ## At end of plotting, reset to previous settings:
       par(op)

       ## Alternatively,
       op <- par(no.readonly = TRUE)                         # the whole list of settable par's.
       ## do lots of plotting and par(.) calls, then reset:
       par(op)
       ## Note this is not in general good practice

       par("ylog")                                           # FALSE
       plot(1 : 12, log = "y")                               # 似乎是设置ylog 参数？？？
       par("ylog")                                           # TRUE

       plot(1:2, xaxs = "i") # 'inner axis' w/o extra space       设置x 坐标轴风格为i 
       par(c("usr", "xaxp"))

       barplot(rbind(nr.prof)) # R 0.63.2: shows alignment problem     #      直方图
       par(las = 0)                                          # reset to default 0 表示平行 ？？但是上面的直方图没有变化？

       require(grDevices) # for gray

                                                                             # bty  是设置 box 的类型，，7表示半封闭
       ex <- function() {                                                    # 参数设置的一个定制函数
             old.par <- par(no.readonly = TRUE) # all par settings which
                                      # could be changed.
             on.exit(par(old.par))
             ## ...
             ## ... do lots of par() settings and plots
             ## ...
             invisible() #-- now,  par(old.par)  will be executed
       }
       ex()

       ## Line types                                  # 设置线的类型的函数
       showLty <- function(ltys, xoff = 0, ...)    {
           stopifnot((n <- length(ltys)) >= 1)

           op <- par(mar = rep(.5,4));
           on.exit(par(op))

           plot(0:1, 0:1, type = "n", axes = FALSE, ann = FALSE)

           y <- (n:1)/(n+1)
           clty <- as.character(ltys)
           mytext <- function(x, y, txt)  text(x, y, txt, adj = c(0, -.3), cex = 0.8, ...)
           abline(h = y, lty = ltys, ...); mytext(xoff, y, clty)
           y <- y - 1/(3*(n+1))
           abline(h = y, lty = ltys, lwd = 2, ...)
           mytext(1/8+xoff, y, paste(clty," lwd = 2"))
       }
       showLty(c("solid", "dashed", "dotted", "dotdash", "longdash", "twodash"))   # 演示各种画线效果
       par(new = TRUE)  # the same:
       showLty(c("solid", "44", "13", "1343", "73", "2262"), xoff = .2, col = 2)
       showLty(c("11", "22", "33", "44",   "12", "13", "14",   "21", "31"))

. 2. plot ---  二维画图函数

  plot 会刷新页面为空白， 然后在其上画图。 plot 之后， 可以使用其他的画图函数进行完善。 

   可以选择画的方式,  "p" 是点，"l" 是线, "b" 是两者, "c" ??  "o" ??  "h"  直方图, "s" stair steps, "S" 其他 steps???

    "n" 没有画图

  plot(x, y, ...)

     x 是 plot 的点坐标。  可以是单个的plot 结构，函数或者R对象

     y  plot 的y 坐标点, 如果x 是是导航的结构， 这个y 是可选的

     ... 是图形参数, 参见par()中的说明  

  例子:

       require(stats)                   # for lowess, rpois, rnorm
       plot(cars)
       lines(lowess(cars))

       plot(sin, -pi, 2*pi) # see ?plot.function

       ## Discrete Distribution Plot:
       plot(table(rpois(100, 5)), type = "h", col = "red", lwd = 10,
               main = "rpois(100, lambda = 5)")

       ## Simple quantiles/ECDF, see ecdf() {library(stats)} for a better one:
       plot(x <- sort(rnorm(47)), type = "s", main = "plot(x, type = \"s\")")         ##  展现内容超出预期
       points(x, cex = .5, col = "dark red")
